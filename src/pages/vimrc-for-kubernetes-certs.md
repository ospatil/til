# vimrc for Kubernetes certs

Add the following to `.vimrc`

```bash
set nu ts=2 sts=2 sw=2 et ic ai si cursorcolumn
" above is same as following
" set number tabstop=2 softtabstop=2 shiftwidth=2 expandtab ignorecase autoindent smartindent cursorcolumn
syntax on
filetype plugin indent on
```
