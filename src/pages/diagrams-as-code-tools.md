# Diagrams as code tools

The main tool for *diagrams as code* is [mermaid](https://mermaid.js.org/) - that allows creating diagrams in markdown,

For architecture diagrams - [https://diagrams.mingrammer.com/](https://diagrams.mingrammer.com/) allows to create diagrams using python.

[https://d2lang.com/](https://d2lang.com/) is another great choice for various types of diagrams.

Icons from the D2 icon list can be used even in diagrams - [https://icons.terrastruct.com](https://icons.terrastruct.com/)
